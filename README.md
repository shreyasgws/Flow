# Flow

A premium checklist app with a hierarchical task tree, dark/light atmospheric mode, and streak tracking.

## Features

- **Tree task model** — tasks can have subtasks with indent/outdent, collapse/expand
- **Dark & Light themes** — dark mode with moonlit mountains, light mode with a flowing warm landscape
- **Dashboard** — overview of overdue tasks, due today, weekly completion chart
- **Smart quick-add** — type `Buy groceries #high @work tomorrow` to set priority, category, and due date in one line
- **Search, filter & sort** — by status, category, priority, due date, or custom order
- **Streak tracking** — tracks daily completion streaks with a 7-day history chart
- **Local storage** — all data persists in the browser
- **Keyboard shortcuts** — `⌘K` command palette, `n` new list, `/` quick add, `Tab`/`Shift+Tab` to indent/outdent
- **Focus mode** — minimize distractions
- **Responsive** — desktop sidebar layout with mobile bottom navigation

## Usage

Open `index.html` in any modern browser. All data is stored locally.

## Tech

Vanilla JS, CSS custom properties, CSS clip-path scenes, canvas particles.
